Overview

Ctrl+Zen is a hackathon prototype that tracks a participant’s attention and emotion from webcam input, then surfaces lightweight signals that can help instructors or facilitators see when a group is drifting, stressed, or engaged.

We scoped the project around a single goal: make it easier to “read the room” in remote workshops without turning the experience into surveillance. That meant aggregating signals, smoothing noise, and keeping the UI simple and ambient.

Tech stack & architecture

  • Used Python + Flask to expose a simple API for attention and emotion scores.
  • Ran OpenCV for face/eye detection and DeepFace for emotion inference.
  • Served a lightweight web dashboard that visualized trends over time with minimal latency.
  • Prototyped flows in Figma first, then implemented the final UI to match that spec.

Highlights

  • Got a full end-to-end pipeline working in a single weekend (capture → inference → visualization).
  • Smoothed raw model outputs so the dashboard felt calm instead of jittery or noisy.
  • Balanced performance and readability so the UI stayed responsive on typical laptops.

What I learned

  • How to glue together ML libraries, APIs, and a browser dashboard under tight time constraints.
  • The importance of privacy-aware design choices for anything that uses webcam data.
  • How to ruthlessly de-scope features so something demoable is ready by judging time.
